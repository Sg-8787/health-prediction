## Health Risk Prediction System
A simple Python project for analyzing health data and predicting health risk levels using basic conditions.

## Features
- Health dataset analysis
- Statistical summary using Pandas
- Data visualization using Matplotlib
- Health risk prediction system
- Beginner-friendly Python project

## Technologies Used
- Python
- Pandas
- Matplotlib
  
## Dataset Parameters
- Age
- BMI(BODY MASS INDEX)
- Sugar Level
- Blood Pressure (BP)
